Linguagem de Marcação

linguagem de marcação usada para estruturar o conteúdo de páginas web.
Ele é a base da web, permitindo que você crie documentos com texto,
imagens, vídeos, links e outros elementos

Íncicio:

É usado "doc" ou "doctype" para
começar um projeto no html

Para conectar com CSS é necessário
usar "link" e para conectar no JS "script"

Tags:

h1-h6: Título
div: Corte
img: Imagem (scr="":Para colocar endereço da imagem)
br: Quebra linha
hr: Gera linha
th: Cabeçario tabela
tr: Coluna tabela
Button: botão
a: Ancora link
p: Paragráfo
ul, ol, li: Representam listas não ordenadas, ordenadas e itens de lista, respectivamente.

"<>" Abre "" Fecha

Atributos de div:

class: Usado quando tem vários elementos
id: Usado quando é apensas um elemento
Clique aqui: Acessar atividade de uso de div

Atributos button

Atributo Descrição type: Define o tipo do botão (button, submit, reset)
name: Nome do botão, usado em formulários
value: Valor enviado ao servidor quando o botão é clicado
disabled: Desativa o botão
autofocus: Define que o botão deve ter o foco ao carregar a página
form: Especifica o formulário ao qual o botão pertence
formaction: URL para enviar o formulário
formmethod: Método HTTP (GET ou POST) para enviar o formulário
formtarget: Especifica onde abrir a resposta do formulário
title: Texto que aparece como dica (tooltip) quando o usuário passa o mouse
