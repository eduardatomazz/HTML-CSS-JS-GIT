Introdução:

O desenvolvimento web envolve várias tecnologias essenciais para criar sites e aplicativos.
O HTML é a base, organizando o conteúdo da página com textos, imagens e links. O CSS é
o responsável por deixar a página bonita, controlando as cores, fontes e layout.
Para manter o código organizado e colaborar com outras pessoas, usamos Git e GitHub,
que ajudam a controlar as versões e o histórico do projeto. E, por último, o JavaScript
torna a página interativa, permitindo que ela reaja a cliques, movimentos do mouse e outras ações do usuário.
